N O N - C O P Y R I G H T - A T T A C H - F O U R
This is a turn based game, where yellow will go first and red will go second.

Players must alternate turns, and only one token can be dropped in each turn.

On your turn, click on the slot where you would like to place your token, the square selected must be at the bottom of the board, or on top of another token that has already been placed. A token placed cannot overlap a slot that has already been taken by the opposing player.

The game ends when there is a four in a row of the same color.

Whoever played as yellow in the first game will be red for the second game.
